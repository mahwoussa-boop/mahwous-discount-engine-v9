# محرك مهووس للحسم v9.0 (Golden Match Edition) 🚀

منظومة استخبارات تجارية متطورة مصممة لمتجر "مهووس" لاستخراج المنتجات المتوفرة لدى المنافسين وغير المتوفرة في المتجر بدقة **100% (صفر أخطاء)**.

## 🌟 المميزات الرئيسية:
- **المعادلة الذهبية للمطابقة (Weighted Fusion):** توزيع الثقل بين الماركة (30%)، الاسم الصافي (40%)، المواصفات (20%)، والبصمة البصرية (10%).
- **التحقق العكسي الثلاثي (Triple Reverse Lookup):** البحث عن المنتج في المتجر بـ 3 طرق مختلفة قبل اعتباره "فرصة جديدة".
- **طبقة الذكاء الاصطناعي الاقتصادية (L5-LLM):** دمج Gemini 1.5 Flash لتصفية الحالات المحيرة بنظام الدفعات (Batching) مجاناً.
- **نظام المرونة التلقائية (Auto-Resilience):** معالجة أخطاء الاتصال وتجاوز حدود السرعة تلقائياً.
- **فلترة العينات:** استبعاد المنتجات أقل من 10 مل تلقائياً.

## 🛠 هيكل المنظومة:
- `logic.py`: المحرك الأساسي الذي يحتوي على كافة المعادلات والطبقات (L1-L5).
- `app.py`: واجهة المستخدم (Streamlit).
- `runner.py`: سكربت التشغيل السريع ومعالجة البيانات الضخمة.

- **796 مكرر محظور**.
- **0 حالات معلقة** (تمت تصفية الجميع بالذكاء الاصطناعي).
